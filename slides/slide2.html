저희 '멍탐정' 프로젝트는 크게 네 가지 기술 영역으로 구성됩니다.

첫째, 프론트엔드 관리자 페이지입니다.
복잡한 웹 기술 대신, Python 코드만으로 데이터 앱을 빠르게 만들 수 있는 Streamlit을 사용해 관리 및 테스트에 필요한 기능을 신속하게 개발했습니다.

둘째, 서비스의 심장인 백엔드입니다.
현대적이고 빠른 FastAPI 프레임워크를 기반으로, 데이터베이스와의 상호작용은 SQLAlchemy라는 ORM을 통해, DB 스키마 변경은 Alembic으로 안전하게 관리합니다.

셋째, 데이터와 지능을 담당하는 부분입니다.
데이터베이스는 안정성이 검증된 AWS의 관리형 서비스인 PostgreSQL을 사용하고, 핵심 AI 기능은 Google의 강력한 최신 모델인 Gemini API를 활용합니다.

마지막으로, 이 모든 것을 떠받치는 인프라와 배포 자동화입니다.
모든 AWS 인프라는 Terraform을 통해 코드로 관리(IaC)되며, Docker와 GitHub Actions를 통해 개발부터 배포까지의 전 과정이 자동화된 파이프라인으로 구축되었습니다.
