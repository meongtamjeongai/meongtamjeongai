How? Code: "마법처럼" 간단한 UI 개발

Streamlit은 Python 스크립트를 웹 앱으로 즉시 변환해줍니다.
복잡한 프레임워크 문법 없이, 간단한 함수 호출만으로 UI 컴포넌트가 생성됩니다.

🐍 Streamlit 코드 예시 (admin_app.py) 


import streamlit as st
from api import ApiClient # 백엔드 API 호출용 클라이언트

# --- 1. 페이지 제목 설정 ---
st.title("🐶 멍탐정 관리자 페이지")

# --- 2. 텍스트 입력창 생성 ---
user_email = st.text_input("검색할 사용자 이메일")

# --- 3. 버튼 생성 ---
if st.button("사용자 정보 조회"):
    # --- 4. 버튼 클릭 시 API 호출 및 결과 표시 ---
    api = ApiClient()
    user_data = api.get_user_by_email(user_email)
    
    if user_data:
        st.success("사용자 정보를 찾았습니다!")
        st.json(user_data) # JSON 형태로 예쁘게 출력
    else:
        st.error("해당 사용자를 찾을 수 없습니다.")

🖥️ 실제 렌더링 결과

그렇다면 Streamlit 코드는 실제로 어떻게 생겼을까요? 정말 놀랍도록 간단합니다.

왼쪽 코드를 보시면, - st.title() 함수 하나로 웹 페이지의 제목이 만들어집니다. - st.text_input() 함수로 텍스트를 입력받는 상자가 생깁니다. - st.button() 함수로 버튼을 만들고, 이 버튼이 클릭되었을 때 실행될 로직을 `if` 문 안에 작성합니다.

저희는 이 버튼 클릭 로직 안에서, 미리 만들어 둔 백엔드 API 클라이언트를 호출하여 사용자 정보를 조회하고, 그 결과를 st.success() 또는 st.error(), st.json() 같은 함수를 통해 화면에 바로바로 표시했습니다.

이처럼 단 몇 줄의 Python 코드만으로 오른쪽과 같은 인터랙티브한 웹 페이지가 완성됩니다. 이 압도적인 생산성 덕분에 저희는 관리자 페이지 개발에 드는 시간을 최소화하고, 더 중요한 핵심 기능 개발에 집중할 수 있었습니다.
